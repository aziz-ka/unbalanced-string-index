# Find Unbalanced String Index


#### Quick Start

```shell
$ git clone git@github.com:aziz-ka/unbalanced-string-index.git
$ cd unbalanced-string-index
$ yarn
$ yarn start
```


#### Yarn/NPM Commands

|Script|Description|
|---|---|
|`yarn start`|Executes the program in Node REPL|
|`yarn test`|Tests the program|
